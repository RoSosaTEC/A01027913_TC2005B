El elemento anchor es una forma de anclar un enlace, ya sea otra sección de la página web actual o una externa.

El elemento párrafo permite separar una sección específica de texto, ¡como la que estas viendo justo ahora!

Los headers pueden utilizarse como títulos o subtítulos. Estos van en varios tamaños, siendo h1 el más grande y h6 el mas pequeño.

strong - Negritas (Importancia)

El elemento strong es una de dos formas de escribir una sección entre líneas en negritas. Si bien ambas formas no son distintas visualmente, la diferencia es que strong encapsula secciones que sean muy importantes

i - Itálicas (Comillas)

El elemento i es una de dos formas de escribir itálicas. Si bien ambas formas no son distintas visualmente, i es utilizada como comillas

b - Negritas (Resaltado)

La segunda forma de negritas. Si bien no son distintas visualmente, esta forma es utilizada para resaltar palabras en el texto
Ejemplo: Una galleta sabe mejor con chispas y leche!!!

strike - Tachado

El elemento strike se utilizaba en HTML4 para tachar texto. Si bien este elemento ya no funciona en HTML5, hay alternativas como "s"

small - Letra pequeña

El elemento small permite reducir el tamaño de la letra en una línea. Si bien no es obsoleto, hay más y mejores formas de hacer este mismo efecto con CSS
Ejemplo:Yo soy grande y yo pequeño

em - Itlálicas (Énfasis)

La segunda forma para escribir italicas. Si bien no son diferentes visualmente, esta forma es utilizada para darle más fuerza a la sección italizada
Ejemplo: ¡Debes ir a hacer tu tarea ahora!

article - Artículo exportable

El elemento artículo es parecido a section. Sin embargo, este elemento está más pensado a poder enviarse fuera del documento original y aún mantener el contexto del mismo. Por ejemplo, todo este párrafo está dentro de un artículo y puede enviarse si así se desea

aside - Sidelines

El elemento aside permite que haya informacion adicional a la par del bloque que lo cubre. Este elemento permite poner el texto contenido a la izquierda o derecha de la página usando CSS.
